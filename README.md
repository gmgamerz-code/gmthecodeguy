# gmthecodeguy Journal

A premium, ultra-professional cinematic blog built with Next.js App Router, Tailwind CSS, React Three Fiber, @react-three/drei, and post-processing.

## Features

- **Fixed Fullscreen 3D Background** — 8,500+ animated particle nodes forming a living wavy digital terrain
- **Cinematic Scroll-Driven Camera** — The camera flies smoothly along a custom Bezier-style path as you scroll the page
- **High-End Glassmorphism** — Beautiful backdrop-blur cards with perfect text readability
- **Post-Processing** — Subtle Bloom + ACES Filmic Tone Mapping for an emissive, premium look
- **Code Splitting** — 3D Canvas is dynamically imported with `ssr: false` and elegant loading state
- **Framer Motion** — Smooth animations and premium modal experience
- **Dark Cinematic Aesthetic** — Designed for quiet luxury and emotional depth

## Tech Stack

- Next.js 16 (App Router) + TypeScript
- Tailwind CSS 4
- React Three Fiber + @react-three/drei
- @react-three/postprocessing + postprocessing
- Framer Motion
